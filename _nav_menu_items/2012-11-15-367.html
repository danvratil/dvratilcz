Akonadi Google is a project that aims to provide Akonadi resources for accessing Google services like calendar, contacts or tasks from KDE PIM. The resources were recently merged to kdepim-runtime to be part of KDE 4.9. The library that provides Google API implementation is located in LibKGoogle git repository.
